# MK TimeGrid

**World Clock, Time Converter & Focus Timer**

World clock, timezone converter, meeting planner, stopwatch, timer, and Pomodoro sessions.

🔗 **Live**: [https://10-web-time-display.vercel.app](https://10-web-time-display.vercel.app)

## Tech Stack

React 19, TypeScript, Vite 6, Framer Motion

## Getting Started

```bash
npm install
npm run dev
```
